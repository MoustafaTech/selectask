How it works

Three seconds, zero context lost.

Works on top of anything — AI chats, docs, PDFs, code editors, terminals, browsers.

step 1

Hold Ctrl + select

Drag over any text while holding Ctrl. Or select first and double-tap Ctrl, or press Ctrl/Cmd + Shift + Space.

step 2

Ask anything

The popup opens at your cursor with the selection as context. Type a question or hit a one-click action: Explain, Summarize, Translate.

step 3

Read, then Esc

The answer streams in as markdown — code blocks included. Ask follow-ups, or press Esc and you're exactly where you left off.

Why it's different

A popup, not another app to babysit.

Bring your own key

Plug in your Anthropic, OpenAI, or Google key — or point it at any OpenAI-compatible endpoint, including local models via Ollama.

Private by design

No account, no server in the middle, no telemetry. Keys live in a local file on your machine; requests go straight to your provider.

Works everywhere

Any app that can select text is supported. Code, prose, error messages, foreign languages — if you can highlight it, you can ask about it.

Streams like you think

Answers render live as markdown with proper code blocks. Stop generation at any moment with one click.

Follow-ups included

The popup keeps the conversation until you close it, so "why?" and "show an example" work as expected.
